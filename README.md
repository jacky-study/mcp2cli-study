# mcp2cli-study

> 深入研究 MCP to CLI 技术实现 — 将任意 MCP/OpenAPI/GraphQL 服务转为命令行工具

## 项目简介

mcp2cli 实现了将 MCP、OpenAPI 或 GraphQL 服务在运行时转换为 CLI 工具，无需代码生成。本项目通过源码阅读和实验，深入理解其架构设计与核心实现。

## 目录结构

```
mcp2cli-study/
├── mcp2cli/         # mcp2cli 源码
├── notes/           # 学习笔记
└── scripts/         # 实验脚本
```

## 学习内容

- MCP 协议原理与实现
- 运行时 CLI 生成技术
- 零代码生成（zero codegen）架构
- API 到命令行的映射策略
